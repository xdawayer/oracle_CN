# 付费体系优化与整合设计方案

> 日期: 2026-03-02
> 状态: 待实施

## 一、设计目标

将现有混合付费模式（订阅 + 积分消费 + 每周配额 + VIP赠积分）简化为两种清晰的付费模式：

- **订阅会员** — 解锁日常使用的所有功能
- **积分购买报告** — 仅用于深度报告类内容

核心原则：**让用户容易理解 — 订阅管日常，积分买报告**。

---

## 二、内容权益矩阵

### 免费内容（所有用户）

| 功能 | 说明 |
|-----|------|
| 深度解读 | 核心人格解读等 |
| 日运分析 | 每日运势 |
| 星盘查看 | 本命星盘 |
| 人生 K 线 | 完整版免费开放 |
| 合盘分析（基础） | 保持现有免费次数 |

### 订阅专属内容

| 功能 | 非订阅用户 | 订阅用户 |
|-----|-----------|---------|
| AI 问答 | 每周免费额度，超出后**引导订阅** | **无限免费** |
| 合盘分析 | 保持现有次数限制，超出后**引导订阅** | **无限免费** |

**关键变化**：
- 非订阅用户超出 AI 问答 / 合盘免费次数后，不再支持积分付费，统一引导订阅
- 移除 AI 问答的 50 积分/次付费路径

### 报告类内容（积分付费，订阅用户也需积分）

| 报告类型 | 价格（积分） | 说明 |
|---------|-----------|------|
| 2026 年度运势报告 | **300** | 全年运势深度分析 |
| 月度深度解读 | **200** | 约 3000-5000 字 |
| 爱情专题深度报告 | **200** | 约 5000-8000 字 |
| 事业专题深度报告 | **200** | 约 5000-8000 字 |
| 财富专题深度报告 | **200** | 约 5000-8000 字 |

**关键变化**：
- 月度报告从 60 → 200 积分
- 专题报告从 150 → 200 积分（统一定价）
- 年度报告 300 积分（新增类型或沿用现有）
- 人生 K 线从 300 积分 → 免费
- 积分的**唯一用途**就是购买报告

## 三、订阅定价

### 价格体系

| 档位 | 首次订阅价 | 续费价 | 现有价格 |
|-----|----------|-------|---------|
| 月度 | ¥9.9 | ¥18 | ¥9.9 |
| 季度 | ¥30 | ¥45 | ¥45 |
| 年度 | ¥168 | ¥198 | ¥128 |

### 规则

- **取消 7 天免费体验**
- **取消订阅赠积分**（现有：月赠100 / 季赠350 / 年赠1500）
- **首次优惠按档位独立判定**：每个档位（月/季/年）有独立的「首次」状态
  - 例：用户首次购买月度 ¥9.9，续费月度 ¥18；此时首次购买季度仍为 ¥30

### 权益列表（更新后）

| 权益 | 说明 |
|-----|------|
| AI 问答无限制 | 不限次数，随时提问 |
| 合盘分析无限制 | 不限次数，多对象分析 |
| AI 深度心理分析 | 结合个人特质的深度分析 |
| 实时周期分析 | 掌握每日洞察指南 |
| 高级关系分析 | 多维度关系互动深度解析 |
| 专属隐私保护 | 数据加密，仅你可见 |

**移除的权益**：
- ~~无限次深度解读~~（现在所有用户免费）
- ~~订阅赠积分~~（取消）

## 四、积分充值体系

### 充值档位

| 充值金额(¥) | 普通积分 | 首充积分(双倍) |
|------------|---------|--------------|
| 10 | 100 | 200 |
| 30 | 300 | 600 |
| 50 | 500 | 1,000 |
| 100 | 1,000 | 2,000 |
| 200 | 2,000 | 4,000 |
| 500 | 5,000 | 10,000 |

### 规则

- **比例 1:10**（1 元 = 10 积分）
- **首充双倍**：用户首次充值（全局判定，非按档位）所有档位均享双倍
- 积分永久有效，不过期
- 积分唯一用途：购买报告

### 与现有体系对比

| 维度 | 现有 | 新方案 |
|-----|------|-------|
| 档位 | 60/100/200/500/1200 积分 | ¥10/30/50/100/200/500 |
| 比例 | 1:10 | 1:10（不变） |
| 首充优惠 | 仅部分档位 | **全部档位首充双倍** |
| 积分用途 | 报告 + AI问答 + 深度解读 | **仅报告** |

## 六、用户体验流程

### 非订阅用户

```
基础内容（免费）→ 使用 AI 问答/合盘
  → 免费次数内 → 正常使用
  → 超出免费次数 → 弹窗引导订阅（不提供积分付费选项）

购买报告 → 选择报告类型
  → 积分充足 → 扣减积分，生成报告
  → 积分不足 → 弹窗引导充值积分
```

### 订阅用户

```
所有日常内容（免费）→ AI 问答/合盘无限使用

购买报告 → 选择报告类型
  → 积分充足 → 扣减积分，生成报告
  → 积分不足 → 弹窗引导充值积分
```

### 付费心智模型

```
用户只需理解两件事：
1. 订阅 = 解锁日常所有功能（AI问答、合盘无限）
2. 报告 = 用积分购买（订阅用户也需要积分）
```

## 七、数据迁移注意事项

- 已订阅用户的赠送积分：已到账的积分保留，停止后续赠送
- 已购买的报告：不受影响
- 现有积分余额：保留，可继续用于购买报告
- K 线报告：已付费用户不受影响，新用户直接免费
- AI 问答积分扣费：立即停止，已扣积分不退还
