# Change: 优化星座配对页面 UI 与功能

## Why

当前星座配对页面（pairing）存在以下问题：
1. **界面风格不统一**：标题布局与其他页面（如 daily、self）不一致，缺乏统一的视觉语言
2. **爱情主题不突出**：作为配对功能，缺乏浪漫、爱情风格的视觉元素
3. **信息展示不完整**：中央卡片只显示星座名称，缺少属相信息的直观展示
4. **中央装饰元素单调**：使用"对比"文字而非更具情感表达的爱心符号
5. **AI 解读能力不足**：当前使用本地算法生成简单文案，缺乏深度占星解读

## What Changes

### UI 层面
- **标题布局统一**：将标题移至页面顶部，采用与 daily/self 页面一致的 header 结构
- **爱情风格强化**：
  - 中央装饰使用 ❤️ 爱心符号替代"对比"文字
  - 整体配色偏向暖色调（玫瑰色、粉色系）
  - 添加浪漫氛围的视觉元素
- **中央卡片增强**：
  - 主体显示星座名称（大字体）
  - 下方显示属相信息（小字体，比例约 3:1）
  - 双方信息对称展示

### 功能层面
- **AI 深度解读**：接入后端 AI 服务，生成专业占星配对分析
- **Prompt 优化**：使用资深占星咨询师角色设定，提供：
  - 命格总论
  - 天赋资源分析
  - 人生考验解读
  - 具体行动建议
  - 开运指南

## Impact

- **Affected specs**: pairing-ui (新增)
- **Affected code**:
  - `miniprogram/pages/pairing/pairing.wxml` - 页面结构重构
  - `miniprogram/pages/pairing/pairing.wxss` - 样式重写
  - `miniprogram/pages/pairing/pairing.js` - 逻辑增强
  - `backend/src/prompts/` - 新增配对解读 prompt（可选）
