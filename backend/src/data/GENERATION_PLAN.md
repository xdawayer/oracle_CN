# 经典书籍深度解读报告生成计划

## 📋 项目概览

**目标**：为"百科-经典"模块生成50本书×2种语言=100份10000字深度解读报告

## ✅ 已完成工作

### 1. 增强版Prompt模板
**文件**：`/backend/src/data/wiki-classic-prompts-enhanced.ts`

**特点**：
- 扩展为7个主要模块，每个模块约1500-2000字
- 总字数目标：约10000字
- 包含详细的小节分解（每个主要部分下有3-4个子部分）
- 中英文版本完整
- 专为DeepSeek Reasoner模型优化，temperature=0.2

### 2. Prompt结构（7模块 × 10000字）

| 模块 | 字数 | 内容要点 |
|------|------|----------|
| 1. 全局定位与背景 | ~1500字 | 书籍地位、作者背景、核心贡献 |
| 2. 核心哲学与理论基石 | ~1500字 | 底层逻辑、核心概念、通俗比喻 |
| 3. 结构化深度导读 | ~2000字 | 逻辑脉络、模块拆解、重点章节 |
| 4. 方法论与实操工具 | ~1500字 | Step-by-Step方法、实用工具 |
| 5. 经典名句与深层解读 | ~1000字 | 金句摘录、思想提炼 |
| 6. 批判性思考与局限 | ~1000字 | 时代局限、争议、阅读误区 |
| 7. 读者行动指南 | ~1500字 | 分阶段学习计划、立即行动 |

## 🚀 执行步骤

### 步骤1：配置环境变量
```bash
# 在 backend/.env 或项目根目录 .env.local 中添加
DEEPSEEK_API_KEY=your_api_key_here
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

### 步骤2：运行生成脚本
```bash
cd backend/src/data
npx tsx generate-enhanced-reports.ts
```

### 步骤3：输出文件
- **个人报告**：`generated-reports/[book_id]_zh.json` / `[book_id]_en.json`
- **汇总数据**：`wiki-classics-enhanced.ts`
